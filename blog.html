1. What is the Document Object Model (DOM)?

Document Object Model (DOM):

  • A representation of an HTML document as a tree structure.
  • Each element, attribute, and text node in the HTML document is represented as an object in the DOM.
  • JavaScript can interact with the DOM to dynamically modify the content, structure, and style of the page.

2. How do you select an element from the DOM?

Selecting an Element:

  • getElementById: Selects an element by its unique ID attribute.
  • getElementsByTagName: Selects elements by their tag name.
  • getElementsByClassName: Selects elements by their class name.
  • querySelector: Selects the first element that matches a CSS selector.
3. What is event delegation in the context of the DOM, and why is it useful?

Event Delegation:

  • A technique where a single event handler is attached to a parent element, and the event is propagated to the appropriate child element.
  • This is useful for handling events on dynamically created elements or when you need to avoid attaching multiple event handlers to individual elements.
4. How do you manipulate an element's attributes and styles using the DOM?

Manipulating Attributes and Styles:

  • Attributes:
    getAttribute: Retrieves the value of an attribute.
    setAttribute: Sets the value of an attribute.
    removeAttribute: Removes an attribute.
  • Styles: style.property: Accesses or sets a style property directly.
    classList: Adds, removes, or toggles classes for styling.
    getComputedStyle: Retrieves the computed style of an element, including inherited styles.
